Invitación web editable para Víctor y Yessica

Cómo usar:
1. Abre index.html en el navegador.
2. Cambia fecha, hora, iglesia, recepción y enlaces en index.html.
3. Reemplaza las imágenes en la carpeta assets:
   - portada.jpg
   - foto1.jpg
   - foto2.jpg
   - foto3.jpg
   - musica.mp3
4. Para RSVP real, crea una cuenta en Formspree y reemplaza REEMPLAZA_AQUI.
5. Para publicarla gratis, sube la carpeta a Netlify, Vercel o GitHub Pages.
